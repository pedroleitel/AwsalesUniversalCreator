# Perguntas para Cliente - Suporte Falcão das Milhas

Objetivo: levantar apenas informações que não apareceram com segurança nas FAQs, snapshots ou tickets. As respostas abaixo devem virar fonte oficial para atualizar a base de conhecimento e reduzir handoffs por `AI_DONT_KNOW`.

## 1. Links e canais oficiais

1. Qual é o link oficial de acesso ao Buscador Automático?
2. Existe link oficial de área de membros, Cademi ou plataforma que a IA pode enviar?
3. Qual canal oficial deve ser usado quando o cliente pede suporte da Oner?
4. Qual canal oficial deve ser usado para acesso ao grupo do Balcão de Milhas?
5. Existe canal oficial para suporte do grupo de promoções?
6. Quando a IA deve encaminhar para humano dentro da plataforma e quando deve orientar outro canal?

## 2. Preço, teste e compra

1. Existe opção de teste do Buscador Automático?
2. Se existe, qual é o valor, duração e condição do teste?
3. Quando o cliente pergunta “como faço o pagamento?”, qual orientação oficial a IA deve dar?
4. A IA pode enviar link de compra/pagamento? Se sim, qual?
5. O que dizer quando o cliente recebeu e-mail inicial sem instrução de cobrança ou pagamento?

## 3. Benefícios, bônus e pacote comprado

1. Quais são exatamente os benefícios inclusos no pacote atual?
2. Alguns clientes mencionaram “8 benefícios”. Quais são esses 8 itens?
3. Onde o cliente acessa cada benefício?
4. Quais bônus existem hoje e como o cliente deve acessá-los?
5. Se o bônus não aparece, qual é o procedimento oficial antes de transferir?

## 4. Acesso, login e segurança

1. Quando a recuperação de senha não chega nem no spam, qual é o procedimento oficial?
2. A IA deve coletar apenas e-mail de compra ou também outro dado? Qual dado é permitido?
3. Em caso de celular roubado/perdido e perda de acesso, qual orientação segura a IA deve dar?
4. Para erro “não foi possível concluir o cadastro”, há algum passo técnico antes do humano?
5. Existe SLA ou frase oficial para casos de ajuste manual de acesso?

## 5. Cancelamento, reembolso e renovação

1. Qual é a política oficial de cancelamento dentro dos 7 dias?
2. Qual é a política oficial de cancelamento fora dos 7 dias?
3. A IA pode mencionar a garantia de 7 dias? Em quais condições?
4. Depois que o cliente preenche o formulário de reembolso, o que acontece?
5. Existe prazo oficial para análise do formulário?
6. Se o cliente diz que já preencheu o formulário e não teve retorno, qual é o procedimento?
7. Como explicar a renovação automática de forma oficial?
8. Em cobrança duplicada, contestação, chargeback ou suspeita de fraude, qual orientação a IA deve dar antes do humano?

## 6. Produtos e direcionamento

1. Quando indicar The Travel?
2. Quando indicar Black Falcon?
3. Quando indicar Balcão de Milhas?
4. Quando indicar Consultoria Individual?
5. Quando uma cotação específica tem taxa?
6. A IA pode informar valores de taxa de cotação? Se sim, quais?
7. Qual é a diferença oficial entre cotação, emissão, suporte operacional e orientação de uso do Buscador?

## 7. Passagem emitida e suporte operacional

1. Se o cliente comprou passagem errada, qual canal ou procedimento oficial?
2. Se o cliente precisa alterar, remarcar ou cancelar passagem já emitida, qual orientação?
3. Se o cliente comprou pelo app/buscador e não localiza a reserva, qual procedimento?
4. A IA pode pedir código localizador? Se sim, em quais casos?

## 8. Perguntas para priorizar

As perguntas mais urgentes para reduzir handoff agora são:

1. Link oficial de acesso ao Buscador.
2. Valor/condição de teste do Buscador.
3. Lista dos 8 benefícios do pacote.
4. Procedimento depois do formulário de reembolso preenchido.
5. Política oficial de renovação e cancelamento fora dos 7 dias.
6. Canais oficiais de Oner, Balcão de Milhas e grupo de promoções.
